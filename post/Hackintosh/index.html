Hackintosh-OpenCore 0.6.3-Big Sur 11.0.1

1.安装U盘制作

因为主力使用Macbookpro所以基于MacOS下进行操作：

  1. Big Sur 11.0.1环境使用 AppStore下载 Big Sur正式版镜像;
  2. 使用磁盘工具格式化U盘,格式：Mac OS扩展(日志式),方案：GUID 分区图 (自带EFI分区);
  3. 使用语句制作Big Sur启动盘（MyVolume为U盘名称）。
sudo /Volumes/Install\ macOS\ Big\ Sur/Install\ macOS\ Big\ Sur.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume

等代码跑完就有了一个Big Sur 11.0.1 的安装U盘。

2.EFI

使用EFI及主板bios设置参考 GeQ1an Hackintosh正常引导

3.硬件

硬件 型号
主板 微星 B360M 迫击炮
CPU 英特尔酷睿 i5-8500
GPU 公版 RX 380 2GB
SSD 三星 PM961 256GB
RAM 海盗船 8GB DDR4 2400MHz x 4

4.环境

环境 版本
OpenCore 0.6.3
MacOS Big Sur 11.0.1
BIOS 版本 7B23v1B

5.替换

  1. 产生未使用的新的三码进行替换；
  2. 根据 GeQ1an 的提示更改代码；
  3. 替换 GeQ1an 提供的 EFI 替换现有U盘的EFI分区。

至此就有了一个带OpenCore 0.6.3的Big Sur 11.0.1 的安装U盘。

6.Hackintosh安装

MacOS 引导安装完成后挂载系统EFI磁盘，将U盘内EFI文件夹替换到系统EFI磁盘内，Hackintosh就可以自OpenCore引导启动。
